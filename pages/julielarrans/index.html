GIF MAIN

utiliser une série de photo avec Gimp, Inkscape, puis Photoscape pour créer un gif.

Ouvrir la première photo dans Gimp
- détourer l'élement : outil > outils de séléction > choisir l'outil qu'on préfère.
- Copier coller l'objet détouré dans un nouveau document.
- utiliser l'outils courbe pour ajouter du constraste pour faciliter la vectorisation : couleur > courbe.
- enregistrer l'image.

Ouvrir l'image modifié dans inkscape pour vectoriser l'objet
- chemin > vectoriser un objet matriciel > choisir le seuil de luminosité.
- exporter en PNG > fichier > exporter au format PNG > choisir le nom et le dossier > exporte.

Reouvrir l'image dans Gimp pour apporter des modification (fermer l'image, ajouter du blanc ou du noir) ou y apporter une couleur
- boite a outils > outils de remplissage > choisir une couleur > cliquer avec le pot de peinture sur le fond.
- renregistrer l'image > exporter.
-répeter toutes les oppérations pour chaques images.

Ouvrir photoscape pour créerle gif
-Créer gif animé > faire glisser toutes les images dans la page > changer la vitesse si besoin (plus le chiffre est grand, plus le gif est lent)
-enregistrer

Gif terminé !
