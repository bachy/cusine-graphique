example de paragraphe

Pour réaliser cette recette, il vous faudra utiliser le logiciel processing, gimp, nodebox et texturing. Pour commencer, il faut générer une image à partir de processing pour obtenir des formes rondes et aléatoires. Générer une image sur nodebox à partir d’un carré et d’une duplication de carrés. Importer l’image crée sur processing ainsi que celle de nodebox sur le logiciel gimp. Jouer avec les calques dans le but d’obtenir l’image souhaitée. Enregistrer l’image et la transférer dans texturing, puis jouer avec la trame. Enregistrer l’image sur texturing, la réimporter sur gimp pour pouvoir jouer avec les couleurs et les calques et arriver à l’image finale.
